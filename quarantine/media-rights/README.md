# Media rights quarantine

Files in this directory are not part of the deployed `public` tree and must not be referenced by rendered pages, metadata, feeds, search data, or sitemaps.

The Nintendo media set was retired on 2026-07-21 because source attribution existed but permission for commercial ad-supported reuse had not been verified. A file may leave quarantine only after its content record has an accepted cleared-rights status and a direct HTTPS evidence URL.

Accepted cleared-rights statuses are `owned-original`, `licensed`, `open-license`, and `public-domain`.
